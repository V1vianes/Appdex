# APPDEX

![Appdex](./images/APPDEX.png)

## Funcionalidades

- Busca de Pokémon.
- Exibição de dados (altura, peso, tipo) e sua imagem.

## Tecnologias

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

- **HTML**
- **CSS**
- **JavaScript**
- **PokeAPI**: Fonte de dados sobre Pokémon.
